## 已收录

### paper-reading — 论文研读笔记

基于 S. Keshav 三遍阅读法（Three-Pass Method）的结构化论文研读 Skill。

- **触发**：`/paper-reading`、`/读论文`、「帮我读这篇论文」
- **流程**：Pass 0（阅读前校准）→ Pass 1（五 C 鸟瞰）→ Pass 2（精读：论证链 + 灵魂图 + 跨领域桥接）→ Pass 3（深潜批判 + 虚拟复现）
- **特色**：阅读目的驱动深度、中文语境适配、跨学科支持
- **来源**：Keshav, S. (2007). How to Read a Paper. *ACM SIGCOMM CCR*, 37(3).
- **课程参考**：[Stanford EE384m Handout](https://web.stanford.edu/class/ee384m/Handouts/HowtoReadPaper.pdf)
