### markdown-to-wechat-html — 微信公众号排版自适应生成

动态读取 Markdown 文章的语境与情感主题，自适应推导出一套专属排版设计系统，输出完全带有内联 CSS 样式的 HTML 文件，供直接全选复制到微信公众号编辑器中。

- **触发**：`/markdown-to-wechat-html`、「帮我把这篇文章转成公众号格式」
- **流程**：题材情感剖析 → 专属配色与排版参数推导 → 微信 HTML 元素样式映射 → 内联 CSS 输出 → 健壮性自审
- **特色**：完全开放的审美推导机制、每篇文章量身定制的视觉系统、100% 内联样式兼容微信编辑器
- **适用**：技术文章、行业分析、散文等 Markdown 到公众号的格式转换
