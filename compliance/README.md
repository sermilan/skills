## 已收录

### policy-decomposition — 政策文件 COU 拆解

将中国网络安全与数据保护政策文件按 COU（合规义务单元）方法论进行结构化拆解。

- **触发**：`/policy-decomposition`、`/拆解政策`、「拆解该政策文件」「按COU规则拆解」
- **流程**：文档层级判定 → COU-R 五要素逐条提取 → 场景聚合分析 → 可视化 HTML 报告
- **输出**：COU-R 原始义务单元（54+ 条/文件）、场景聚合文件（5–8 个/文件）、HTML 合规拆解报告
- **特色**：五要素义务原子化（主体/动作/客体/条件/权重）、权重量化（base_weight × factor）、合规工作流场景聚合
